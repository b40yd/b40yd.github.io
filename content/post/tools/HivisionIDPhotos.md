# HivisionIDPhoto 证件照制作

HivisionIDPhoto 旨在开发一种实用的证件照智能制作算法。

它利用一套完善的模型工作流程，实现对多种用户拍照场景的识别、抠图与证件照生成。

HivisionIDPhoto 可以做到：

- 轻量级抠图（仅需 CPU 即可快速推理）
- 根据不同尺寸规格生成不同的标准证件照、六寸排版照
- 美颜（waiting）
- 智能换正装（waiting）
